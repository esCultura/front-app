## Dependencies

- Node >= v16.13.2
- npm >= v8.1.2

## Building and Running Locally
#### Clone the repo

```
git clone https://github.com/esCultura/front-app.git
```

#### Install Dependencies
Execute the following commands to install all the necessary dependencies
```shell
cd esCultura
npm install
```
